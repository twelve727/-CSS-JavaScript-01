화면 배치 방법(형식) : display 속성

        요소가 화면에 어떻게 보여질지 형식을 지정하는 속성

        - block : 화면을 수직 분할(행 나눔)
            + width / height 넓이 / 높이 속성 사용 O

        - inline : 화면을 수평 분할(하나의 행 컬럼을 나눔 == 글자)
            + width / height 넓이 / 높이 속성 사용 X --> 수평이니까

        - inline-block : 수평 분할 + block 크기 조정

        - none : 화면에 요소가 표시되지는 않으나, 존재하는 상태

        - flex : 요소의 [정렬되는 방향], 요소 간의 [간격]을 유연하게 처리하는 형식
